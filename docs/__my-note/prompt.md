自分で調べたいと思った英単語を蓄積していくアプリを作成したいです。
ざっくりと以下のようなイメージです。

* 入力フィールドがあって、そこに調べたい英単語、英語の表現を入力する
* 入力したものをインターネット上から調べて、意味や解説を表示
	* 簡単な例文も自動生成
	* 何で出会った英語なのかをタグ付けすることができる
	* 任意でメモを残せる
	* 調べたものやタグやメモなどはローカルに保持する
* 入力フィールドに入力したものが過去にも調べたことがあるものだった場合は、その情報を引っ張ってきて表示する

これを元に、実案件として社内で使えそうな要件定義としてマークダウンファイルにして保存してください。
